# Palette

這當作一個紀錄

最初的本意是練習使用 react

並想出一個自己可能會使用到的功能並實踐

當時對前端懂得還不多

所以有很多奇怪的寫法

左側的顏色選擇器是在網路上找的

## Getting Started

        npm install

## Running the tests

啟動
        
        npm run electron

打包

        npm run package


## Built With

* Babel - 將新版本的JavaScript編譯成 ES5
* React - 建構 SPA 的前端框架
* electron - 將網頁打包成應用程式
